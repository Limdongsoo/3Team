ava Spring Boot(Spring Boot) :

세 가지 핵심 기능을 통해 Spring Framework를 사용하여 
더 빠르고 쉽게 웹 애플리케이션과 마이크로서비스를 
개발하도록 돕는 툴입니다.


spring-boot-starter-web:

스프링 MVC를 사용해 RESTful이 포함된 
웹애플리케이션을 작성하기 위한 스타터다. 기본 임베디드 컨테이너로 톰캣을 사용한다.


spring-boot-devtools:

클래스 경로의 파일이 변경될 때마다 자동으로 재시작해준다.
즉, 코드 변경에 대한 매우 빠른 피드백을 제공한다.
기본적으로 디렉토리를 가리키는 클래스 경로의 모든 항목은 변경 사항에 대해 모니터링 된다.


Spring Security:

Spring 기반의 애플리케이션의 보안(인증과 권한, 인가 등)을 담당하는 스프링 하위 프레임워크다.
Spring Security는 인증과 권한에 대한 부분을 Filter 흐름에 따라 처리하고 있다.



타임리프(Thymeleaf):

자바 라이브러리이며, 웹과 환경 양쪽에서 TEXT , HTML , XML , Javascript , CSS 를 생성할 수 있는 템플릿 엔진이다.
스프링 MVC와의 통합 모듈을 제공하며, 애플리케이션에서 JSP로 만든 기능들을 완전히 대체할 수 있다.



JPA:

객체를 데이터베이스의 하나 이상의 테이블로 매핑하기 위해 주석 또는 XML을 사용하여 객체-관계형 매핑의 중요한 작업을 표준하는 기술이다.


Lombok:

여러가지 @어노테이션을 제공하고 이를 기반으로 반복 소스코드를 컴파일 과정에서 생성해주는 방식으로 동작하는 라이브러리이다.
즉, 코딩 과정에서 롭복과 관련된 어노테이션만 보이고 getter,setter 등의 생략되지만 실제로 컴파일된 결과물


QueryDSL:

QueryDSL은 하이버네이트 쿼리 언어(HQL: Hibernate Query Language)의 쿼리를 타입에 안전하게 생성 및 관리해주는 프레임워크이다.
QueryDSL은 정적 타입을 이용하여 SQL과 같은 쿼리를 생성할 수 있게 해 준다.


MariaDB:

커뮤니티에서 개발하여 상업적으로 지원되는 MySQL 관계형 데이터베이스 관리 시스템(RDBMS)의 포크입니다.
MariaDB의 로그와 측정항목을 수집하려면 mysql 수신자를 사용합니다.
